# Solution Seeking Queries

This repository stores project files for collecting, refining, and tracking solution-seeking queries.

Source conversation: https://chatgpt.com/g/g-p-69b46c19a7c88191a7ea225f5db1b406-021/c/6a2e8576-a7cc-83ee-bfb0-ff13f1534e8a

## Structure

- `queries/solution-seeking.md` - working query set and notes.
- `templates/query.md` - reusable format for new query entries.
- `docs/process.md` - lightweight workflow for turning a question into an implemented solution.

## Workflow

1. Capture the problem statement clearly.
2. List constraints, assumptions, and available context.
3. Break the question into testable sub-questions.
4. Record candidate solutions and tradeoffs.
5. Track the selected action and verification result.
